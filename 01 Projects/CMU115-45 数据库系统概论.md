# Lecture 1

data model 与 schema 不同
前者是抽象的数学概念，而后者是有关数据存储结构的编码概念

依旧是分层抽象，不要向用户暴露任何有关数据的组织结构的具体实现

1. 原数据
2. 物理结构（数据编码）
3. 逻辑结构（数据表）
4. 虚拟结构（虚表）
5. 用户层

关系型数据库在透明性这方面，甚至有些极端，她连指针都不允许用户染指，完全把用户当作呆瓜
 
primary key 唯一标识一个实体
foreign key 将不同实体间建立关系
